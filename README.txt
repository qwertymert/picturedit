----------------------------------------------------------
In this folder, there are 2 main files;

picturedit.py : The python script that contains the image editing module AND image
editing application. User can run picturedit.py to use it.

Picturedit Module Tutorial.ipynb : A tutorial for using picturedit module 
designed for jupyter notebook.

Required Libraries to run picturedit.py and Picturedit Module Tutorial.ipynb;
- numpy : used for image processing
- PIL  : used for reading and saving images
- tkinter : used for creating graphical user interface
- math : used for trigonometric calculations

Note: picturedit_icon.png is used as icon of picturedit application. It was created by me.
